# Explore: Adobe® Firefly™ or a Similar Generative AI Tool

**Course:** AI Essentials
**Module:** 1
**Type:** Non-graded exploration activity

## Overview
This is a practice/explore activity built for a 100-level Introduction to AI course at a community college. 

A guided, view-only exploration activity that introduces students to generative AI as a distinct AI workload. Students browse Adobe® Firefly™ (or an alternate tool) without creating an account, observe how prompt-based image generation works, and compare generative AI to rule-based and predictive systems. A comparison table, key vocabulary section, and three reveal-answer reflection questions are included.

## File
`index.html` — single self-contained HTML file, no dependencies or external libraries required.

## Instructor Notes
- If Adobe® Firefly™ is unavailable to students, the activity lists three free alternate tools: DALL-E (OpenAI), Canva AI Image Generator, and Ideogram.ai.
- If an alternative tool is chosen/directed, update the reference text and swap out the screenshots with ones appropriate to the alternative tool or remove them completely.
- Reflection questions are for student self-assessment only. No submission is expected.
